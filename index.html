# 介绍

一个兼容 IE8 且使用方式类似 Vue 的框架，学习成本低，使用方便。

Vue 基于 Object.defineProperty 这个浏览器 API（Vue3 改为使用 Proxy）实现了响应式更新，直接变更数据，即可驱动 UI 更新。而由于 IE8 的这个 API 有缺陷，导致不能在 IE8 上使用 Vue。

本框架通过声明及搜索依赖的方式，使用发布订阅模式，驱动 UI 更新。

  • 支持大部分 Vue 指令的功能。除了 v-text，v-pre，v-cloak。

  • 支持组件、slot 插槽、组件私有样式（scoped style）。可以编写组件库，每个组件库可以是同一主题或同一功能的组件集合，按需组合。这些组件可以异步加载、懒加载，并支持在 加载中/加载失败 时，显示指定的 loading/error 组件。

  • 组件支持使用中文命名，可以在模板中用中文引用，丰富对组件的描述。

  • 支持在模板中绑定事件时直接使用箭头函数（不需 babel 编译）。

  • 绑定事件时，冒泡事件会采用委托模式绑定。在 IE8 中会通过模拟捕获阶段来支持绑定捕获阶段事件。

  • 模板支持多个根节点。（注：所有根节点都会自动继承来自父组件的非 prop 属性，该行为与 Vue3 不同。详情）

  • 整合了 Vue 的 <transition> 和 <transition-group> 的功能。使用 <transition> 即可对多个元素添加 enter/leave/move 过渡及设置 mode 过渡模式。详情

  • 提供状态管理工具，与 Vuex 的使用方式有部分类似，进行了一定的简化。

  • 所有用于声明依赖和变更数据的方法或属性，都能以键路径的形式表达要依赖或变更的数据。

  • 已经实现的功能，大部分都参照 Vue3 文档的迁移细节进行修改。例如：调整 if 指令的优先级高于 for 指令、绑定属性时的冲突合并结果由属性次序决定、增加 emits 选项、$attrs 包含 class & style 等。具体请详看本文档。

使用：

  1. 框架内置在 ccb_toolkit_v5.js 中，使用前需先在页面引入 js。



     



     <head>
   ...
   <script src="http://www.ccb.com/cn/v3/js/ccb_toolkit_v5.js"></script>
   ...
 </head>

  2. 使用 toolkit.createApp 方法创建根实例。详细的使用方法请查阅本文档。

    示例：

    toolkit.createApp({
  template:
    '<div class="container">\
      <input $model="search">\
      <transition name="flip-list" appear move mode="out-in">\
        <button $if="showAdd" @click="add">添加</button>\
        <button \
          $else-if="showShuffle"\
          @click=`()=>$setData("list", _.shuffle(list))`\
        >\
          打乱顺序\
        </button>\
        <TextRow $for="text in searchList" @remove="remove" :key="text">\
          {{ text }}\
        </TextRow>\
      </transition>\
    </div>',
  data: function () {
    return {
      search: "",
      list: ["Bruce Lee", "Jackie Chan", "Chuck Norris", "Jet Li", "Kung Fury"]
    }
  },
  computed: {
    searchList: function () {
      var search = this.search

      if (!search) return this.list

      return this.list.filter(function (text) {
        return ~text.indexOf(search)
      })
    },
    showAdd: function () {
      return this.searchList.length === 0
    },
    showShuffle: function () {
      return this.searchList.length > 1
    }
  },
  methods: {
    add: function () {
      this.$setData({
        "list": this.list.concat(this.search),
        "search": ""
      })
    },
    remove: function () {
      var removeText = this.text,
        newList = this.list.filter(function (text) {
          return text !== removeText
        })

      this.$setData("list", newList)
    }
  },
  style: {
    ".container": {
      "margin": "10px",
      "position": "relative",
      "display": "flex",
      "flex-direction": "column"
    },
    "input": {
      "width": "100%"
    },
    "button": {
      "margin": "5px 0",
      "align-self": "flex-end",

      "&.flip-list-enter": {
        "transform": "translateX(30px)",
      },
      "&.flip-list-enter-active": {
        "transition": "opacity 1s,transform 1s"
      }
    },
    ".flip-list-move": {
      "transition": "transform 1s"
    },
    ".flip-list-enter,.flip-list-leave-to": {
      "opacity": "0"
    },
    ".flip-list-enter-active,.flip-list-leave-active": {
      "transition": "opacity 1s",

      "&.flip-list-move": {
        "transition": "opacity 1s,transform 1s"
      }
    },
    ".flip-list-leave-active": {
      "position": "absolute"
    }
  },
  components: {
    TextRow: TextRow
  },
  el: "#app"
})

function TextRow() {
  return {
    template:
      '<li ><slot></slot><button @click=`$emit("remove")`>删除</button></li>',
    emits: ["remove"],
    style: {
      "li": {
        "cursor": "default",
        "display": "flex",
        "justify-content": "space-between",
        "align-items": "center",
        "margin": "5px 0",
        "width": "100%",

        "&:nth-child(odd)": {
          "color": "red"
        },
        "&:nth-child(even)": {
          "color": "green"
        },
        "button": {
          "margin-left": "5px",
          "flex-shrink": "0"
        }
      }
    }
  }
}

# 键路径

键路径是用于描述数据的字符串。它的写法与 js 从对象/数组中获取数据时的写法一致。

var data = {
    user: { name: 'John', age: 30, sex: 'man' },
    firends: [
        { name: 'Jack', age: 31, sex: 'man' },
        { name: 'Rose', age: 25, sex: 'woman' }
    ]
}

从上面对象获取 name 属性的键路径写法：

'user.name'
'user["name"]'
'firends[0].name'
'firends[0]["name"]'

键路径支持在以下位置使用：

  • 组件实例的 $setData 方法

     


     


      this.$setData('user.name', name)

  this.$setData({
    'user.name': name
  })

  • 组件实例的 $watch 方法

      this.$watch('user.name', function(){})

  • 组件的 watch 选项



     



    {
  watch:{
    'user.name': function(){}
  }
}

  • 状态管理工具的 provide 方法

    store.provide('user.name')

  • 状态管理工具的 commit 方法

    store.commit('user.name', name)
